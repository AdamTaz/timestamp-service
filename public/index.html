Timestamp Microservice

Please use this API by submitting a unix time (in ms) or a natural date string as a parameter. See below for an example

Example input

  • http://localhost:8080/getdate/March%2016%202017
  • http://localhost:8080/getdate/1489622400000

Example output

  • {"unix":1489622400000,"natural":"Thu Mar 16 2017"}
